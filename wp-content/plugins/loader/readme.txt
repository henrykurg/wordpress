Loader is a lightweight plugin to show preloader and preload pages on mouse hover to load pages faster.

== Description ==

[Loader](https://broadbrander.com/loader) is an user experience booster plugin. No one likes waiting, so create beautiful preloader screens with unlimited random quotes to let them read something while page is loading and stick user on your website for longer and reduce bounce rate.


The plugin allows you to use:

* Beautiful CSS preloaders.
* Image as preloader.
* Random Quotes on preloader screen.
* Prefetch pages on mouse hover (Helps to load page faster).


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/loader` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Appearence -> Customize -> Loader Panel screen to configure the plugin.


== Frequently Asked Questions ==

= Can I disable the quotes on preloader screen? =

Yes you can do this from Appearence -> Customize -> Loader Panel -> Settings. 

= Is it compatible with page builder? =

Yes! You can use Loader with page builders.
